getMeshes!~YUIDOC_LINE~!* @param object

() Array

Returns a array of THREE.Mesh instances from the given object. If nested transformations are found, they are applied to child meshes as mesh.userData.matrix, so that each mesh has its position/rotation/scale independently of all of its parents except the top-level object.

Returns:

Array:

meshes found inside the Object3D
