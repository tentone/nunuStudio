Features

Graphical Editor

What you see is what you get. Familiar user interface similar to the ones you already know.

WebVR

Ready for web virtual reality, on booth mobile and desktop platforms.

Physics

3D physics powered by cannon.js

JS Scripts

Easy to learn, no technical knowledge required.

2D Mode

Want 2D? No problem! Switch between 2D and 3D with the click of a button.

Advanced Graphics

Graphics powered by three.js with support for dynamic ligths, shadows, post-processing effects, animations, etc.

Material Editor

Graphical material editor similar to the ones found in 3D design tools.

Particle

GPU accelerated particle system powered by SPE.

Drag and drop

Drag and drop files into the editor, it just works.

Spine animations

Create 2D animations using Esoteric spine animation tool and import them to nunuStudio.

WebAudio

Audio engine built on top of web audio API with support for positional audio.

Multi-platform editor

nunuStudio was build on top of nwjs it works on windows, linux and OS X.

Understands You

Load anything you want into the editor, tons of file formats are supported.

Open Source

nunuStudio is a open source project, everybody can colaborate and help.
